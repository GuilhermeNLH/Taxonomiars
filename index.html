Como usar o Taxonomy Map Builder

Primeiros passos

  1. Acesse o aplicativo em qualquer navegador moderno — não é necessário instalar nada.
  2. Escolha um modelo em Configurar → Modelos (Membranas CO₂, Biogás ou Mapa em Branco).
  3. No painel Adicionar, preencha os campos e clique em:
    • Atualizar Central — define o tema principal do mapa.
    • Adicionar Driver Meso — cria grupos (com cor, nome, descrição).
    • Adicionar Micro — cria subitens dentro de um Meso.
    • Adicionar Artigo — adiciona referências ligadas a um Meso.
  4. Clique ou toque em qualquer nó no mapa para editá-lo; use o painel Nós para listar, abrir rapidamente e excluir.
  5. Exporte via botões do topo (SVG, HTML, JSON ou Excel) ou importe um JSON salvo para continuar a edição.

Painel lateral

  • Adicionar: cria o nó central, mesos, micros e artigos.
  • Nós: lista hierárquica com contadores; clique/toque abre a edição. O "X" remove o nó e seus descendentes.
  • Configurar: edita Título, Subtítulo e Query; ajusta espaçamento; alterna exibição de artigos; carrega modelos.

Edição de nós

Clique com o mouse ou toque rápido sobre qualquer bloco do SVG para abrir o modal de edição. No modal você pode alterar nome, descrição, cor, DOI/URL e driver pai. O botão Deletar remove o nó e todos os seus filhos.

Navegação e zoom

  • Desktop: arraste com o botão esquerdo para mover; roda do mouse para zoom. Botões flutuantes: ＋/－ (zoom), ⊡ (reset), ⤢ (ajustar ao quadro).
  • Touch: arraste com um dedo para mover; pinça para zoom. Toque rápido em um nó abre a edição.

Importar e exportar

  • Exportar SVG: baixa a arte vetorial limpa.
  • Exportar HTML: gera um HTML autossuficiente com o SVG embutido.
  • Exportar JSON: salva o estado completo do mapa.
  • Exportar Excel: gera uma planilha com central, mesos, micros e artigos (inclui DOI/URL clicável).
  • Importar JSON: recarrega um arquivo salvo (o mapa atual é substituído).

Novo mapa

O botão ＋ Novo Mapa abre uma confirmação e reseta o estado para um mapa vazio.

Notas rápidas

  • O SVG é recalculado a cada alteração; o zoom não muda a geometria, só a visualização.
  • A coluna de artigos pode ser ocultada em Configurar → Mostrar artigos.
  • O aplicativo é 100% client-side — basta o arquivo HTML e um navegador moderno.
  • DOI/URLs dos artigos viram links clicáveis no mapa e na lista.
